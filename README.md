 📘 Digital Skills Gap Management Information System (MIS)

## 📌 Overview

The **Digital Skills Gap Management Information System (MIS)** is a web-based platform designed to assess users' digital competencies, identify skill gaps, and provide personalized learning recommendations.

The system helps individuals and organizations understand current skill levels and bridge gaps through structured guidance and targeted learning resources.

---

## 🎯 Objectives

* Assess users' digital skill levels through structured tests
* Identify gaps in knowledge and competencies
* Provide tailored learning recommendations
* Track user progress over time
* Support decision-making for training and development

---

## 🛠️ Features

* 👤 **User Registration & Login**
* 📝 **Skill Assessment Tests** (Beginner, Intermediate, Advanced)
* 📊 **Automated Skill Gap Analysis**
* 🎯 **Personalized Recommendations**
* 📈 **Progress Tracking Dashboard**
* 🗂️ **Admin Panel for Managing Questions and Users**

---

## 🧠 How It Works

1. Users create an account and log in
2. They take a digital skills assessment test
3. The system evaluates responses
4. Skill gaps are identified automatically
5. Personalized recommendations are generated
6. Users track their improvement over time

---

## 🗄️ Database Structure

Key tables include:

* Users
* Assessments
* Questions
* Results
* Recommendations

1. Install a local server (e.g., XAMPP or WAMP)
2. Clone or download the project files
3. Place the project in the `htdocs` directory
4. Import the database into MySQL
5. Configure database connection in the project files
6. Run the project via `localhost`
